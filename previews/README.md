# Скриншоты Phoenix LMS

Для добавления скриншотов сделайте следующее:

1. Сделайте скриншоты каждой страницы
2. Сохраните их в эту папку с именами:
   - `login.png` — страница входа
   - `dashboard.png` — дашборд студента
   - `courses.png` — каталог курсов
   - `course-view.png` — просмотр курса
   - `teacher-dashboard.png` — панель преподавателя
   - `mega-menu.png` — мега-меню навигации

## Рекомендации по скриншотам

- Размер: 1920x1080 или меньше ( пропорции 16:9 )
- Формат: PNG или JPG
- Сделайте скриншоты в браузере при разрешении 1280x720+

## Пример команды для скриншота (Linux)

```bash
# Скриншот окна браузера
gnome-screenshot -a -f login.png
```
